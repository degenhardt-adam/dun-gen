Prerequisites:
- Python 2.7 or newer (https://www.python.org/downloads/ or from the Microsoft Store)

To run:
- Download dun-gen.py
- Open a terminal
    - On Windows: hit Win+r, type cmd, hit enter
- Navigate to where you downloaded dun-gen.py
    - On Windows, type (without the quotes) "cd .\Downloads" (or the path to wherever you saved the file)
- Type (without quotes) "python dun-gen.py"
- If it does something different from what you think it should do, DM me on discord
